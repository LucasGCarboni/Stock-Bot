# Política de Privacidade — Stock-Bot

## 1. Quem somos

O Stock-Bot é mantido por desenvolvedores independentes, você pode consultar o código-fonte ou repositório no [GitHub](https://github.com/LucasGCarboni/Stock-Bot?tab=coc-ov-file).  
Este documento explica como coletamos, usamos, armazenamos e protegemos os dados dos usuários.

## 2. Dados que podemos coletar

Dependendo das funcionalidades usadas, podem ser coletados:

- ID do usuário Discord, nome de usuário, servidor e canal onde o usuário interage com o Bot.
- Dados enviados pelo usuário ao usar comandos (ex: tickers, símbolos, texto).
- Dados retornados por APIs externas (preço de ações, dados de mercado, etc).
- Informações de logs ou armazenamento temporário para performance ou métricas.

**Não coletamos** dados sensíveis como e-mail, telefone, dados bancários ou informações pessoais além do necessário para o funcionamento do Bot.

## 3. Finalidade do uso dos dados

Utilizamos os dados para:

- Processar comandos e retornar informações no Discord.
- Melhorar a aplicação, detectar erros, otimizar desempenho.
- Diagnóstico e depuração, logs internos em caso de falhas.
- Possivelmente estatísticas anônimas de uso (sem identificação pessoal), para entender como o Bot está sendo usado.

## 4. Compartilhamento de dados

- Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros.
- Podemos divulgar dados se exigido por lei ou autoridade competente.
- Dados de APIs externas são consultados em tempo real; não há garantia de armazenamento permanente.

## 5. Armazenamento e segurança

- Dados são armazenados de forma segura, com acesso restrito aos desenvolvedores do Bot.
- Caso haja persistência (por exemplo logs ou histórico), será mantido por tempo limitado, apenas o necessário para o funcionamento, correção ou auditoria.
- Se você solicitar, é possível remover quaisquer dados vinculados ao seu ID (se houver armazenamento).

## 6. Consentimento e retirada

Ao usar o Bot você consente com a coleta e uso de dados conforme descrito aqui.  
Se, a qualquer momento, desejar que seus dados sejam removidos, entre em contato conforme informado na seção de contato.

## 7. Mudanças nesta Política

Podemos alterar esta Política de Privacidade a qualquer momento, a versão atual estará disponível em `docs/privacy_policy.md`.  
Recomendamos a verificação periódica da política.
